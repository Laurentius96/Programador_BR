The overflow Property

A propriedade overflow especifica se o conteúdo deve ser recortado ou adicionado barras de rolagem quando o conteúdo de um elemento é muito grande para caber em um determinado área.

The overflow property specifies whether to clip content or to add scrollbars when an element's content is too big to fit in a specified area.
